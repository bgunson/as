This is AdShare

AdShare allows anyone to host ads themselves. Sharing is caring, which means you will host your own ads as well as others ads. This of course goes both ways, as your ads will be replicated to others hosting ads.

Installation Guide

Step 1

First step is to navigate to a folder or spot on your computer you would like to set this up. Then open a console such as git bash and type the following command:

git clone https://github.com/bgunson/as.git

Step 2

Next use the command:

cd as/peer/frontend && npm install && npm start build && cd .. && npm install

to install all the necessary node components. To run the peer client now, just type npm start which will launch a peer instance as well as a local site you can use to manage ads.
